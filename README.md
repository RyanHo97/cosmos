# cosmos.css手册

**cosmos**（宇宙）  

## 项目介绍

Creating CSS style libraries for individuals,Easy to use.

为个人创建CSS样式库，易于使用

整体风格基调为暗色

## 框架特点

- 免费开源
- 纯CSS轻量化
- 支持响应式的web界面
- 具备主流浏览器兼容性

## 使用方式

简单导入即可使用：

```html
<link rel="stylesheet" href="./static/cosmos.css">
```

如需生产环境版本，请等待后续的更新。
